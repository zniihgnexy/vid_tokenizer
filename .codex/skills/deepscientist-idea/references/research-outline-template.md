# Research Outline Template

Use this reference when the `idea` stage needs a durable research-outline style document before selecting a final direction.

Prefer these section headers:

```md
# Executive Summary

# Codebase Analysis

# Dataset Analysis

# Mathematical Problem Formulation

# Baseline Methods as Special Case Solutions

# Five Actionable Research Directions

# Evaluation Metrics and Success Criteria

# Research Infrastructure and Constraints
```

## Section expectations

- `Executive Summary`
  - purpose
  - current results
  - dataset characteristics
  - core algorithm summary
  - major limitations
  - performance gap analysis when possible
- `Codebase Analysis`
  - major modules and files
  - data flow
  - key classes and functions
  - dependencies
  - computational hotspots
  - current evaluation pipeline
  - improvement potential rating per baseline
- `Mathematical Problem Formulation`
  - rigorous notation
  - objective and constraints
  - complexity or theoretical limits when relevant
  - `3-5` decomposed sub-problems
- `Baseline Methods as Special Case Solutions`
  - mathematical characterization
  - constraint analysis
  - theoretical gaps
  - extension potential
- `Five Actionable Research Directions`
  - exactly five directions when the space is broad enough
  - each direction should include both the challenge and the proposed approach
  - balance exploration cost against expected improvement
```
