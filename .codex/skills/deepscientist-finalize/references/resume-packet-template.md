# Resume Packet Template

Use this when the quest is pausing rather than ending permanently.

## Recommended structure

### 1. Current state

- quest objective
- present stage
- final recommendation at pause time

### 2. Accepted baseline

- baseline id or path
- status of reproducibility / comparability
- key metric context

### 3. Strongest evidence

- top 2-3 results
- supporting artifact paths
- key caveats

### 4. Open blockers

- blocker
- why it matters
- what stage should resolve it

### 5. Next action

- the single best next step
- why it dominates alternatives
- what should be read first before acting

### 6. Do-not-repeat notes

- failed branches not worth retrying unchanged
- misleading metrics or comparability traps
- known environment or tooling pitfalls
